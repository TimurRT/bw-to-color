# bw-to-color

Раскрашивание Ч/Б изображений с помощью вариационного автоэнкодера

Проект использует torchvision - https://pypi.org/project/torchvision/

Цель модели — восстанавливать цветовую информацию из чёрно-белых изображений.
Модель получает на вход канал L (яркость) из цветового пространства Lab и предсказывает каналы ab (цветовые компоненты), которые затем объединяются для восстановления полного RGB-изображения.

Для начальных экспериментов используется датасет CIFAR-10.
